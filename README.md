## :computer: For Devs

1. Clone the repository...
```
git clone https://github.com/NASA-SUITS-techteam/TSS.git
```
3. Navigate to `/frontend`
4. Run...
```
npm i
```
5. Run...
```
npm start
```
6. In another terminal navigate to `/backend`
7. Run...
```
npm i
```
8. Run...
```
npm run dev
```
6. Open in your browser
```
localhost:4200
```
