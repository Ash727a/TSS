## :computer: For Devs

1. Clone the repository...
```
git clone https://github.com/NASA-SUITS-techteam/TSS.git
```
3. Navigate to the `/frontend` or `/backend` folder
4. Run...
```
npm i
```
5. Run...
```
npm start
```
